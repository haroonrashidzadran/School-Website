## 🎯 About The Project

**Khair Un Nass Private High School** website is a comprehensive, professional educational institution website that showcases the school's identity, academic programs, admissions process, and contact information.

This website is designed to be:
- ✅ Fully responsive (mobile, tablet, desktop)
- ✅ Easy to customize
- ✅ SEO-friendly
- ✅ Accessible and user-friendly

---

## ✨ Features

### Core Features
- **Responsive Navigation** - Fixed navbar with mobile hamburger menu
- **Hero Sections** - Full-width hero with parallax effect and professional Unsplash images
- **Statistics Counter** - Animated number counting on scroll
- **FAQ Accordion** - Expandable/collapsible FAQ items
- **Contact Forms** - Client-side validated forms
- **Smooth Scrolling** - Anchor link navigation with smooth scroll

### Pages
1. **Home Page** - Hero, welcome section, features, statistics, programs, testimonials, CTA
2. **About Page** - History timeline, mission & vision, principal's message, facilities, team
3. **Academics Page** - Academic levels, subjects, extracurricular activities, calendar
4. **Admissions Page** - Admission process, requirements, fee structure, application form, FAQ
5. **Contact Page** - Contact information, office hours, map, contact form

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Code editor (VS Code recommended)
- Basic knowledge of HTML/CSS/JS

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/haroonrashidzadran/School-Website.git
   ```

2. **Navigate to project directory:**
   ```bash
   cd School-Website
   ```

3. **Open in browser:**
   - Simply double-click `index.html`
   - Or use a local server (recommended for full functionality)

4. **Using VS Code Live Server:**
   - Install "Live Server" extension
   - Right-click `index.html` → "Open with Live Server"

## 🎨 Customization

### Changing Colors

Edit the CSS variables in `css/styles.css`:

```css
:root {
    --primary: #1a365d;      /* Change primary color */
    --secondary: #c9a227;   /* Change accent color */
    --accent: #2d5a3d;      /* Change accent color */
    /* ... more variables */
}
```

### Changing Images

Replace Unsplash URLs in each HTML file with your own images:
```html
<!-- Example -->
<img src="YOUR_IMAGE_PATH" alt="Description">
```

### Modifying Content

Simply edit the HTML files to update text, links, and content as needed.

## 📦 Deployment

### GitHub Pages (Recommended)

1. **Push to GitHub:**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/haroonrashidzadran/School-Website.git
   git push -u origin main
   ```

2. **Enable GitHub Pages:**
   - Go to Repository Settings
   - Navigate to "Pages" (under "Code and automation")
   - Select Source: "Deploy from a branch"
   - Select Branch: "main" and folder: "/ (root)"
   - Click Save

3. **Your site will be live at:**
   ```
   https://haroonrashidzadran.github.io/School-Website/
   ```
